# 零号遗嘱 · PROTOCOL ZERO

> 当死者要求你证明他仍然活着。

纯前端叙事推理游戏：你是数字遗产管理局的校验员，在 180 分钟的执行窗口内审讯死者的 AI 人格镜像「零号」，通过档案比对、证词固定、证据连接与时间线重建，裁决它是「林默本人」「独立人格」还是「工具」——9 种结局。

## 运行

无需构建。直接用浏览器打开 `index.html`，或：

```bash
npx serve .
```

## 在线演绎（可选，LLM 接入）

游戏默认使用内置剧本引擎，离线可完整通关。接入 LLM 后解锁：规则未命中的自由提问演绎、基于真实审讯记录生成的第四幕画像、引用你实际言行的最终陈词。

```bash
cp api.config.example.js api.config.js   # 填入你的 Key（OpenAI 兼容接口）
```

也可在游戏内「设置 → 在线演绎」中直接配置。架构原则：**规则引擎做大脑，LLM 做声带**——分幕知识白名单 + 分幕禁令 + 输出黑名单校验，保证解谜公平性不被模型泄露破坏。

## 文件

| 文件 | 说明 |
|---|---|
| `index.html` / `style.css` | 界面与「数字遗产法医主义」设计系统 |
| `data.js` | 真相底稿：档案、证据、连接白名单、剧本规则、结局 |
| `game.js` | 引擎：意图解析 / 证词流 / 质证 / 时间线 / 画像 / 裁决 / Canvas 人格视觉 / 合成音频 |
| `llm.js` | LLM 演绎层（可选） |
| `零号遗嘱_剧情文本.md` | 全部剧情文本底稿 |
| `零号遗嘱_攻略.md` | 完整攻略（含结局矩阵） |
| `_validate.js` | 数据一致性校验：`node _validate.js` |
