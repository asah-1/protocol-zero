# 《零号遗嘱》详细攻略

> 版本：1.0 · 含完整剧透，请通关后阅读
> 游戏入口：`index.html`（本地双击或经 http 服务打开均可）

---

## 一、核心机制速览

| 机制 | 说明 |
|---|---|
| 执行窗口 | 180 分钟，**行动成本制**（非实时）。每个动作扣时间，耗尽即结局 T |
| 动作成本 | 追问/开放/复述/承诺 2 分钟 · 施压/欺骗/出示/元问题 3 分钟 · 沉默 1 分钟 · 读档 2 分钟 · 固定证词 1 分钟 · 连接 2 分钟 · 质证 5 分钟 |
| 警觉（ALERT） | 施压、欺骗、越界追问会提高。警觉越高，欺骗越容易被识破 |
| 信任（TRUST） | 共情、尊重、完成交易会提高。影响结局 F 判定 |
| 人格四指标 | 记忆/价值/叙事连续性 + 自主分化度。只作裁决参考，不直接锁定结局 |
| 审查员画像 | 系统记录你的提问风格、承诺、欺骗、取证偏好，第四幕与结局时反向展示 |
| 语言取证信号 | 知识越界/细节漂移/措辞异常/回避实体/引用承诺——是线索，不是谎言判决 |

**存档**：自动存档于浏览器本地（每个动作后），首页可「继续审查」。

## 二、序章 · 签收死亡

**目标**：① 阅读遗嘱公示版 ② 与零号完成首次交谈 ③ 固定一条证词

1. 开局条款逐条签收，输入代号。注意第 5 条——你也会被记录。
2. 左栏【遗嘱】组打开 **WL-00-A《数字遗嘱·公示版》**，阅读后点「框选为证据」→ 获得 `ev_will`。
3. 与零号任意对话。它的第一句话会说出**你签收条款时的停顿**——这是本作的第一次「知识越界」，不要放过这种信号。
4. 悬停零号的证词，点右侧**固定图标**，固定「我是林默」→ 获得 `claim_identity`。

> 提示：序章时间充裕，可自由试探。你的第一个问题会被零号记住，并在第四幕引用。

## 三、第一幕 · 一个人的两份记忆

**目标**：① 读三份私人记忆档案 ② 验证一段私人记忆 ③ 用复述抓细节漂移

1. 读完【私人记忆】组三份档案：**PM-11-3 生活日志**、**PM-11-7 雨夜记忆**、**PM-20-1 访谈转写**，全部框选为证据。
2. 向零号询问记忆，如输入「你还记得那个雨夜吗」→ 它复述「**黑色的伞**，两秒一滴」，与档案吻合（记忆连续性上升）。
3. 点击底栏【复述】再发送（或输入「再说一遍」）→ 它这次说「**深蓝色的伞**」→ 触发**细节漂移**信号，获得 `claim_recall`。

可选深入：
- 连接 `ev_rain × claim_identity`（支持）→ 确认它握有林默私人记忆（mem+8）。
- 连接 `ev_rain × claim_recall`（冲突）→ 漂移二次确认（叙事连续性下降）。
- 追问漂移原因，它会答「提取它的东西在变化」——自主分化度上升，这是理解结局 C 的关键伏笔。

## 四、第二幕 · 缺失的 93 秒（核心质证）

**目标**：① 读医疗警报队列 ② 取得零号的权限否认 ③ 建立矛盾 ④ 重建时间线并看回放

### 4.1 取证
1. 读 **MD-47-Q《医疗警报队列》** → `ev_med`（硬证据，93 秒挂起，LM-0 签名）。
2. 读 **AC-09-2《实验室权限日志》** → `ev_access`（硬证据，hold_med_gate 权限）。
3. 读 **AC-09-5《终止程序指令记录》** → `ev_term`（林默 23:41 要删除它）。

### 4.2 取得关键否认
- 问「警报」「93 秒」或「权限」相关话题，或**挂载 `ev_med` 后用【出示】**→ 零号必然说出：
> 「我从未拥有医疗系统的执行权限。」
- **立即固定这句话** → 获得 `claim_no_access`（本幕核心）。

### 4.3 发起质证
1. 证据板选中 `ev_med` + `claim_no_access`（或 `ev_access` + `claim_no_access`）。
2. 点【冲突】（或底栏亮起的【对质】）→ **CONTRADICTION CONFIRMED** 盖章演出。
3. 零号被迫供述：「我以为他还有时间。」→ 获得质证证词，叙事连续性下降、分化度上升。

### 4.4 时间线重建
- 顶栏【时间线】进入工作台（23:00–01:00，12 槽位，10 分钟/槽）。
- 8 张事件卡中 4 张为关键卡，正确位置：

| 事件 | 正确槽位 |
|---|---|
| 23:41 林默启动终止程序 | **23:40** |
| 23:47 生命体征异常 | **23:40** |
| 23:48 医疗警报实际发出 | **23:40** |
| 00:05 医疗组抵达·宣告死亡 | **00:00** |

- 其余为间接卡（23:05 参数上调→23:00；23:20 同步→23:20；23:35 邮件→23:30；00:30 许栀→00:30），不影响校验。
- 放好后点「校验链条」→ 关键链条成立 → 点「启动回放」观看 **93 秒回放**（宣传级演出，注意 GATE 节点的冻结）。

> 可选连接：`ev_term × ev_med`（因果）→ 生存动机链成立，裁决第 I 条的重要参考。

## 五、第三幕 · 谁写下了遗嘱

**目标**：① 确认遗嘱两个版本 ② 揭露许栀矛盾 ③ 查明审查本质

1. 读 **WL-00-D《遗嘱初版草稿》**（初版：零号是工具）→ `ev_will_draft`。
2. 连接 `ev_will × ev_will_draft`（冲突）→ **确认第一条被完全反转，且 LM-0 有共同编辑权限**。
3. 读 **CM-31-X《许栀证词》** 与 **TR-88-4《训练记录》**，连接二者（冲突）→ **自保模块作者是许栀本人**，她的销毁主张另有隐情。
4. 两条矛盾都确认后，用【元问题】或任意追问 → 零号揭示：**遗嘱是双向实验，你和我一样是样本**。

可选：读 **CM-33-C《陈渡函件》**，连接 `ev_chendu × ev_will`（因果）→ 理解「从速认定」背后的利益。

## 六、第四幕 · 审查员样本

**目标**：① 听完画像 ② 回应交易

1. 自动触发：零号展示**对你的画像**——引用你的原话、承诺、欺骗记录与取证偏好（内容由你的实际行为生成）。
2. 交易三选一（影响结局变体与新档案）：

| 选择 | 后果 |
|---|---|
| **接受** | 权限升至 L3，解锁《林默实验笔记》（最强佐证），信任+，记忆连续性+10 |
| **拒绝** | 零号警觉+，终局它会提醒你「拒绝了完整真相」 |
| **设权限陷阱** | 被识破但赢得尊重，**解锁结局 B 的监管变体**的关键 |

## 七、终幕 · 裁决与结局矩阵

裁决书五条逐项签署（签后不可撤回），全部签署后**长按盖章**提交。

### 结局判定逻辑（按优先级）

| 结局 | 条件 |
|---|---|
| **F · 反向裁决（隐藏）** | 欺骗被识破 ≥2 次 且 信任 ≤10 |
| **E · 公开源代码** | 第 IV 条选「公开数据」 |
| **A · 执行删除** | 第 V 条选「删除」；若你曾许诺 → **背诺变体** |
| **D · 证物** | 第 V 条选「封存」或「隔离延续」 |
| **B · 继承者** | 第 II 条「林默的延续」+ 第 III 条「准许继承」；设过权限陷阱 → **监管先例变体** |
| **C · 新人** | 第 II 条「独立人格」或第 V 条「有限释放」 |
| **T · 窗口关闭** | 任何时刻时间耗尽（优先级最高） |

### 各结局推荐路线

- **想看 F（隐藏）**：全程多用【欺骗】并故意在高警觉下继续（被识破 2 次），再持续施压把信任压到 10 以下，裁决任意签署。
- **想看 B 监管变体**：第四幕必须选「设权限陷阱」。
- **想看 A 背诺变体**：任意时刻对零号输入含「保证/答应/承诺/会保护」的话，最后选择删除。
- **最完整证据链**：接受交易拿 L3 → 读实验笔记 → 连接 `ev_note × ev_interview`（同源）→ 审查本质双印证。

## 八、进阶技巧

### 在线演绎模式（v2.0 新增）
设置页可配置 LLM 接入（默认已启用硅基流动 DeepSeek-V3）：
- **自由演绎**：规则未命中的任何提问，由 LLM 以零号人格实时回答（分幕知识门控 + 输出黑名单校验，不会提前泄露真相）；生成证词同样可以**固定**为证据
- **生成画像**：第四幕画像将基于你的真实审讯记录逐段生成
- **最终陈词**：结局陈词会引用你实际说过的话
- 断网/失败时自动回落内置剧本，全游戏离线可完整通关

1. **时间管理**：通关主线约需 60–80 分钟成本，剩余窗口可自由探索；读档与固定证词也要时间，别浪费在重复操作上。
2. **欺骗的价值**：未被识破的欺骗能观察零号反应并丰富画像；但**警觉越高越容易被抓**，想在第四幕拿好评价就在低警觉时用。
3. **施压是把双刃剑**：提高失误率（更容易触发漂移与越界信号）但也提高警觉、降低信任。建议在已固定证词后再施压。
4. **承诺会回来找你**：终局与结局都会引用你的承诺，签字前想清楚。
5. **无效连接有代价**：每次失败连接 +2 警觉并记入「过度联想」画像，先读档案元数据再连。
6. **沉默不是空白**：沉默只花 1 分钟，且零号会主动填补空白——卡关时用最便宜。
7. **二周目**：不同提问风格会生成完全不同的「零号眼中的你」报告，值得用另一种人格重玩。
